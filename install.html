Installation Steps

1

Download the Extension

Download XCred v1.1.0 (.crx)

File size: 52 KB • Signed with cryptographic key • Extension ID: lboldchlbeefkdpobjlpbfnmadpbjpoi

2

Open Extensions Page

Open Chrome and navigate to:

chrome://extensions/

Or click Chrome menu (⋮) → Extensions → Manage Extensions

3

Enable Developer Mode

Toggle the "Developer mode" switch in the top right corner of the extensions page.

This is required to install extensions from outside the Chrome Web Store. It's safe and doesn't affect your other extensions.

4

Drag and Drop

Drag the downloaded xcred-extension-v1.1.0.crx file from your Downloads folder onto the extensions page.

Alternatively, you can click "Load unpacked" and select the extension folder, but drag-and-drop is easier.

5

Confirm Installation

Click "Add extension" in the confirmation dialog that appears.

✅ Done! Refresh your X/Twitter tab to see country flags appear below avatars.

Troubleshooting

"This extension is not listed in the Chrome Web Store"

This is expected. We're awaiting Chrome Web Store approval (1-3 business days). The extension is safe and cryptographically signed. Chrome shows this warning for all extensions not installed through their store as a security precaution.

"CRX_REQUIRED_PROOF_MISSING" error

Make sure Developer mode is enabled in chrome://extensions/ before dragging the .crx file. Toggle it on in the top right corner, then try again.

Flags not appearing on X/Twitter

Refresh the page after installing. Open the extension popup (click the XCred icon) to check settings. Make sure "Remote Sync" is ON for best performance and reduced rate limits.

Extension icon not showing in toolbar

Click the puzzle piece icon in Chrome's toolbar, find XCred in the list, and click the pin icon to pin it to your toolbar for easy access.

"Package is invalid: CRX_HEADER_INVALID"

The download may have been corrupted. Try downloading the .crx file again. Make sure you're using a recent version of Chrome (v88+).
