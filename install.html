Installation Steps

2

Extract the ZIP File

Right-click the downloaded .zip file and select "Extract All..."

Remember the location where you extract it (e.g., Downloads folder). You'll need this folder in the next step.

3

Open Extensions Page

Open Chrome and navigate to: chrome://extensions/

Or click Chrome menu (⋮) → Extensions → Manage Extensions

4

Enable Developer Mode

Toggle the "Developer mode" switch in the top right corner of the extensions page.

This is required to install extensions from outside the Chrome Web Store. It's safe and doesn't affect your other extensions.

5

Load Unpacked Extension

Click the "Load unpacked" button that appears after enabling Developer mode.

Navigate to and select the extracted folder from Step 2 (the folder containing manifest.json).

Make sure you select the folder itself, not the .zip file or a file inside the folder.

6

Verify Installation

XCred should now appear in your extensions list with a blue/green flag icon.

✅ Done! Refresh your X/Twitter tab to see country flags appear below avatars.

Important: Don't delete the extracted folder after installation. Chrome needs it to run the extension.

Troubleshooting

"This extension is not listed in the Chrome Web Store"

This is expected. We're awaiting Chrome Web Store approval (1-3 business days). The extension is safe. Chrome shows this warning for all extensions not installed through their store as a security precaution.

"Load unpacked" button is grayed out

Make sure Developer mode is enabled in chrome://extensions/. Toggle it on in the top right corner, and the "Load unpacked" button will become active.

Can't find the extracted folder

Check your Downloads folder for the extracted XCred folder. If you can't find it, re-extract the .zip file and note the location. The folder should contain files like manifest.json and content.js.

Flags not appearing on X/Twitter

Refresh the page after installing. Open the extension popup (click the XCred icon) to check settings. Make sure "Remote Sync" is ON for best performance and reduced rate limits.

Extension icon not showing in toolbar

Click the puzzle piece icon in Chrome's toolbar, find XCred in the list, and click the pin icon to pin it to your toolbar for easy access.

"Manifest file is missing or unreadable" error

Make sure you selected the folder containing the extension files, not the .zip file itself. The folder should have manifest.json directly inside it.

I accidentally deleted the extracted folder

The extension will stop working. Re-download the .zip file, extract it to a permanent location (like Documents/Chrome Extensions), and load it again using "Load unpacked".

Updating to a newer version

To update XCred to a new version:

  1. Download the latest .zip file from this page
  2. Extract it to a new folder (or replace the old extracted files)
  3. Go to chrome://extensions/
  4. Find XCred and click Remove to uninstall the old version
  5. Click "Load unpacked" and select the newly extracted folder
  6. Refresh any open X/Twitter tabs

Simply refreshing the extension may not load new files. Removing and re-adding ensures you get the latest version.
