Privacy Policy

Last updated: November 23, 2025

Overview

XCred is a browser extension that displays publicly available account transparency information on X/Twitter. This policy explains what data we access and how we handle it.

Data We Access

XCred accesses the following publicly available information from X/Twitter:

  • Account location ("Account based in")
  • Connection platform ("Connected via" - iOS App Store, Android, Web)
  • VPN/Proxy detection status
  • Verification status (Blue, Business, Government)
  • Account creation date
  • Username change history

This is the same information visible to anyone viewing an account's "About" page on X.

Data We Store

Local Cache (Your Browser): Profile data is cached locally using IndexedDB to reduce API calls and improve performance. This data stays on your device.

Shared Cache (Optional): If "Remote Sync" is enabled in settings, anonymized profile data is shared with other XCred users via our database to reduce X API rate limits. You can disable this anytime.

Data We DO NOT Collect

  • Your X/Twitter login credentials
  • Your personal information
  • Your browsing history
  • Private messages or protected tweets
  • Your IP address or device information
  • Any data from accounts you don't view

Third-Party Services

Supabase: Used for the optional shared profile cache. Stores only public profile transparency data (location, platform, verification status). No personal user data is stored.

We do not use any analytics, tracking, or advertising services.

Data Sharing

  • We do not sell any data
  • We do not share data with advertisers
  • Cached profile data may be shared with other XCred users (if Remote Sync is enabled) to improve performance
  • All shared data is already publicly accessible on X

Your Controls

  • Disable Remote Sync: Turn off shared caching in extension settings
  • Clear Local Cache: Delete all cached data via extension settings
  • Uninstall: Removes all local data and stops all data access

Data Security

Local data is stored in your browser's IndexedDB, protected by your browser's security model. Shared cache data is stored in Supabase with row-level security policies.

Children's Privacy

XCred is not intended for use by children under 13. We do not knowingly collect data from children.

Changes to This Policy

We may update this policy as the extension evolves. Changes will be posted on this page with an updated date.
